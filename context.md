This file serves as the persistent memory for the development of the Kitty Terminal Portfolio. Read this file at the start of any session to quickly synchronize state and requirements.

---

## 🎯 Project Overview
An interactive, high-fidelity terminal emulator portfolio website designed exactly to match the user's personal Linux Kitty Terminal. It is a client-side frontend project designed to teach web development concepts (React, TypeScript, CSS layout models, event handlers) through a gamified series of quests.

---

## 🎨 Visual & Design Specifications
*   **Wallpaper**: Cosmic starry background with planet overlays and blurred nebula gradients.
*   **Terminal Frame**: Rounded corners, thin border, semi-transparent purple-slate body (`rgba(27, 27, 36, 0.85)`), backdrop blur filter, and a Kitty title bar.
*   **Monospace Typography**: `JetBrains Mono` or `Fira Code` imported from Google Fonts.
*   **Two-Line Shell Prompt**:
    *   *Line 1*: `╭─ 󰊠 billa at ~` (left-aligned) and a live ticking clock `🕒 HH:MM` (right-aligned).
    *   *Line 2*: `╰─> ` (followed by active command cursor).
*   **Top-Left Sprite**: Pixel art of Regirock displayed when the console boots or on command trigger.

---

## 🛠️ Tech Stack
*   **Core**: React 18+ (TypeScript)
*   **Tooling**: Vite (Dev Server & Bundler)
*   **Styling**: Pure CSS (CSS Variables for values, Flexbox/Grid for layout structures, Keyframes for animation/glitches).
